## 数据驱动
性能优化基于数据驱动，一般情况有两个指标可衡量：业务数据、性能数据。

一套完整的性能优化方案应该具有如下能力：

* 分析现状：数据采集能力、性能分析能力
* 实施手段：性能优化手段、AB实验能力
* 拿到结果：数据报表分析能力
* 防止劣化：性能指标监控能力

通过**标准化埋点方案**采集数据，拿到数据后分析性能瓶颈找出优化产出比最高的点重点处理，通过AB实验做基准对比，待拿到业务数据或者性能数据时分析是否呈现正向反馈来继续扩量，直至全量后，拿到完整的性能优化产出。

建设线上指标监控，防止因业务变动导致的性能指标劣化。
