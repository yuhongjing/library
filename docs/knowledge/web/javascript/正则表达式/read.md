## 每个章节讨论了什么？

### 第一章和第二章

正则是匹配模式，要么匹配字符，要么匹配位置。



### 第三章

在正则可以使用括号捕获数据，要么在API中进行分组引用，要么在正则里进行反向引用。



### 第四章

学习正则，是需要了解其匹配原理的。（第六章最后一节也讲解了正则表达式的整体工作原理）



### 第五章和第六章

不仅能看懂别人的正则，还要自己会写正则。



### 第七章

学习正则，是为了在真实世界里应用的。
