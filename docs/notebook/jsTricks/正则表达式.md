# Regex Skill
正则表达式相关的JS。


## 获取url的queryParams参数
```js
function formatUrlQuery() {
    const urlParams = window.location.search;
    if (!urlParams) {
        return null;
    }
    const keys = {};
    urlParams.replace(/([^?=&]+)=([^&]*)/g, (full, key, value) => {
        keys[key] = value;
    });
    return keys;
}
```
