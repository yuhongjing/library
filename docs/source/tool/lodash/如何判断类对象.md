# 如何判断类对象

```js
function isObjectLike(value) {
  return typeof value === 'object' && value !== null;
}
```

通过`typeof`操作符，返回值为`object`且值不为`null`，就认为是类对象。

因为`null`的typeof返回`object`，所以需要单独过滤，这是JS的bug，但是未修复。

另外还需要注意一点，在由宿主实现的对象中，规范规定了不能返回 `undefined`、 `boolean`、 `number` 和 `string` 这几种类型，但是 `document.all` 例外，返回的是 `undefined` ，这是不遵循规范的实现。
