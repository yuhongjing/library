# 唐代·陆龟蒙
> 陆龟蒙（？～公元881年），唐代农学家、文学家、道家学者，字鲁望，号天随子、江湖散人、甫里先生，长洲（今苏州）人。

> 曾任湖州、苏州刺史幕僚，后隐居松江甫里（今甪直镇），编著有《甫里先生文集》等。

> 他的小品文主要收在《笠泽丛书》中，现实针对性强，议论也颇精切，如《野庙碑》、《记稻鼠》等。

> 陆龟蒙与皮日休交友，世称“皮陆”，诗以写景咏物为多。
