# 唐代·张旭
> 张旭（675年—约750年），字伯高，一字季明，汉族，唐朝吴县（今江苏苏州）人，开元、天宝时在世，曾任常熟县尉，金吾长史。

> 以草书著名，与李白诗歌，裴旻剑舞，称为“三绝”。诗亦别具一格，以七绝见长，与李白、贺知章等人共列饮中八仙之一。与贺知章、张若虚、包融号称“吴中四士”。书法与怀素齐名。

> 性好酒，据《旧唐书》的记载，每醉后号呼狂走，索笔挥洒，时称张颠。实也说明他对艺术爱好热狂度，被后世尊称为“草圣”。
