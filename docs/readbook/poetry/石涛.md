# 清代·石涛
> 石涛（1642年－1708年），明末清初著名画家，原姓朱，名若极，广西桂林人，祖籍安徽凤阳，小字阿长，别号很多，如大涤子、清湘老人、苦瓜和尚、瞎尊者，法号有元济、原济等。明靖江王朱亨嘉之子。

> 与弘仁、髡残、朱耷合称“清初四僧”。

> 石涛是中国绘画史上一位十分重要的人物，他既是绘画实践的探索者、革新者，又是艺术理论家
