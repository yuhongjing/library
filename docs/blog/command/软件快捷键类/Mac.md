# Mac使用方法整理
整理Mac相关的使用方法。

## 多屏时，如何固定dock的位置
鼠标移入屏幕最底部，dock就会出现在这个屏幕。  
如果无效，多试一下即可。

## 快捷键
Mac与Window不一样的操作快捷键整理

功能|快捷键
:-:|:--
剪切|先`command + c`复制目标文件，再`command + option + v`
显示隐藏文件|`command + shift + .`
快速搜索|`command + 空格`
删除文件|`command + del`
锁屏|`command + option + 关机键`
内置翻译软件|`control + command + d`
