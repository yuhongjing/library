## 移动端控制台
页面引入脚本，即可开启虚拟的控制台。  
插件有很多，腾讯等都有开发。  
```js
<script src="//cdn.jsdelivr.net/npm/eruda"></script>
<script>
    eruda.init();
    console.log('控制台打印信息');
</script>
```

## 手机端访问电脑端本地项目
按照如下方式即可在手机端访问电脑端的项目
1. 手机与电脑处于一个局域网内
2. 电脑开启一个本地端口，例如localhost:8080
3. 查看电脑端的ip地址，例如192.168.0.135
4. 手机访问192.168.0.135:8080即可

::: warning 注意
如果是vue等开启本地服务器时  
需要在package.json script dev 加上 --host 0.0.0.0

:::
