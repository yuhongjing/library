## Mac必备应用汇总
提高Mac开发效率的应用

## 编辑器
* **`Typora Markdown`** Mac系统下优秀的markdown编辑器
* **`vsCode`** 很优秀的前端编辑器
* **`Scivener`** 写作编辑器

## 前端工具
* **`postman`** 功能强大的测试API工具

## 系统
* **`iTerm 2 + on my zsh shell + tmux`** Mac shell全家桶
* **`Homebrew`** Mac必备的下载工具
* **`Alfred 4`** 快速启动器(搜索等)
* **`source code pro`** 程序员必备的字体
* **`Magican Rest`** 定时休息提醒
